How it works

  • Messages are sent via POST https://graph.facebook.com/v23.0/{{ '{' }}PHONE_NUMBER_ID{{ '}' }}/messages
  • The Authorization: Bearer TOKEN header is added automatically.
  • Phone numbers are normalized to E.164 format (+country code + number).
  • Template variables like ${{ '{' }}contactName{{ '}' }} are resolved per contact before sending.
